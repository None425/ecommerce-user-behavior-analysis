# 사용자 로그 데이터 기반 카테고리 매출 성장 구조 분석


## 프로젝트 개요
E-Commerce 사용자 행동 로그 데이터를 분석하여   
구매 전환 퍼널을 정의하고 단계별 이탈 구조를 파악하였다.   
또한 요일 및 카테고리별 전환율 차이를 분석해 전환율 개선 방향을 도출한 프로젝트이다.  

## 활용 데이터 
1. E-커머스 로그 데이터 - https://www.kaggle.com/code/annettecatherinepaul/ecommerce-analysis-and-recommendations/input

## 데이터 구조
- event_time : 이벤트 발생 시각  
- event_type : 사용자 행동 유형(view, cart, purchase, remove_from_cart)  
- product_id : 제품 ID    
- category_id / category_code : 제품 카테고리 정보      
- brand : 제품 브랜드  
- price : 제품 가격         
- user_id : 유저 ID     
- user_session : 유저 세션

## 데이터 분석
### 1. 퍼널 전환 구조 분석
<img width="790" height="490" alt="funnel_rate" src="https://github.com/user-attachments/assets/88c833d5-8ee1-42ea-ac5c-c4acd286bec3" />

|유형|유저수|전환율|이탈율|
| -- | -- | -- | -- |
| View | 358,212 | -- | -- |
| Cart | 72,359 | 20.2% | 79.8% |
| Purchase | 25,290 | 34.95% | 65.05% |

전체 퍼널 구조를 분석한 결과, View → Cart 단계에서 가장 큰 이탈이 발생하는 것으로 나타났다. 상품을 조회한 사용자 중 약 20.2%만이 장바구니로 이동하며, 약 79.8%의 사용자가 초기 단계에서 이탈하고 있다.  
또한 Cart 단계에서도 약 34.95%만이 실제 구매로 이어져, 장바구니 이후 단계에서 약 65%의 추가 이탈이 발생하는 구조로 확인되었다.  
이는 단순히 상품을 둘러보는 관심 단계에서 실제 구매 행동으로 이어지는 과정이 가장 어려운 구간이며, 이 구간의 전환율을 개선하는 것이 전체 구매 전환율을 높이는 데 가장 중요한 요소로 판단된다.  

### 2. 요일별 카테고리별 조회/구매 수  
#### 조회 수
<img width="1790" height="590" alt="week_cat_view" src="https://github.com/user-attachments/assets/adb823c1-5ffc-47aa-8b48-80d33fac8269" />

'기타' 카테고리가 전체 조회의 대부분을 차지하여 세부 카테고리별 패턴이 왜곡되는 현상이 나타났다.  
이에 따라 '기타'를 제외한 분석을 수행한 결과, 모든 주요 카테고리에서 월요일과 일요일에 조회 수가 증가하는 공통 패턴이 확인되었다.  
특히 토요일에는 전반적인 활동 감소가 관찰되었으며, 일요일에 다시 상승하는 구조를 보였다. 이는 주말을 중심으로 한 탐색 및 소비 활동이 강화되는 경향으로 해석될 수 있다.  
#### 구매 수
<img width="1790" height="590" alt="week_cat_pur" src="https://github.com/user-attachments/assets/2a768267-c0ec-4405-81f9-75d353e40811" />

대부분의 카테고리에서 월요일과 일요일에 상대적으로 높은 구매가 발생하고, 주중 중반 및 토요일에 감소하는 패턴이 확인되었다.   
특히 의류와 가전제품 카테고리에서는 일요일에 뚜렷한 반등이 나타났다.   
조회 패턴과 구매 패턴이 유사하게 나타나는 점을 고려할 때, 요일에 따른 사용자 활동성과 구매 행동 간의 연관성이 존재할 가능성이 있다.  

결론
--- 
일요일에는 조회 수와 구매 수가 함께 증가하는 경향이 나타났다.   
이는 단순히 상품을 둘러보는 활동이 늘어나는 것에 그치지 않고, 실제 구매로도 이어질 가능성이 높다는 점을 보여준다.   
반면 토요일은 조회와 구매 모두 상대적으로 낮은 수준을 보이며, 전반적인 활동이 감소하는 구간으로 확인되었다.  
이러한 결과는 요일에 따라 사용자 관심과 구매 행동이 함께 변화하는 경향이 있음을 의미한다.   
특히 일요일은 비교적 높은 활동이 나타나는 시점으로, 프로모션이나 상품 노출을 집중하기에 적절한 시점일 가능성이 있다.   
또한 월요일 역시 활동 수준이 높은 구간으로 나타나, 요일에 따른 전략 수립에 참고할 수 있는 인사이트를 제공한다.  

### 3. 카테고리별 구매 퍼널 전환율 분석
<img width="990" height="590" alt="cat_view_pur" src="https://github.com/user-attachments/assets/99af7414-7f28-42b7-bff1-436758fe344b" />

| 카테고리 | View→Cart | Cart→Purchase | View→Purchase |
| -- | -- | -- | -- |
| 의류 | 87.9% | 46.8% | **41.1%** |
| 문구 | 54.0% | 46.0% | 24.8% |
| 가전제품 | 21.1% | 34.9% | 7.4% |
| 가구 | 22.3% | 26.5% | 5.9% |
| 악세서리 | 8.6% | 31.8% | 2.7% |
| 기타 | 23.6% | 30.7% | 7.3% |

- 의류 카테고리는 조회 대비 구매 전환율이 41.1%로 가장 높게 나타났다.
- 문구 역시 전 단계에서 높은 전환율을 보이며 비교적 효율적인 카테고리로 확인되었다.
- 기타 카테고리는 트래픽 규모는 압도적이나 최종 전환율은 평균 수준으로 나타났다.
- 가전·가구 카테고리는 장바구니 이후 이탈이 발생하는 구조로 보인다.
- 악세서리는 조회 대비 장바구니 전환율이 8.6%로 가장 낮아 초기 단계에서 병목이 발생하고 있다.

카테고리마다 전환이 발생하는 단계와 이탈 구간이 서로 다르게 나타났다.
따라서 카테고리에 동일한 전략을 적용하기보다는, 카테고리 특성에 맞춘 맞춤형 개선 전략이 필요하다

## 분석 결과 및 한계
### 분석 결과
- 전체 사용자 대비 최종 구매 전환율이 낮은 구조로 확인됨  
- View → Cart 구간에서 가장 큰 이탈이 발생하며, 초기 단계가 핵심 병목으로 나타남  
- Cart → Purchase 구간 또한 유의미한 이탈이 존재하는 주요 전환 구간으로 확인됨  
- 카테고리별 구매 전환 구조의 편차가 크게 존재함을 확인  
- 일부 고전환 카테고리가 전체 구매 전환을 견인하는 구조로 분석됨  
### 분석 한계
- 가격, 할인 등 상품 속성 정보는 반영하지 못했다는 점
- 매출·수익성까지는 분석 범위에 포함하지 못했다는 점
- 
## 전략 제안
1. 상품 조회에서 장바구니 단계에서 이탈이 크기 때문에 장바구니 버튼이 더 눈에 띄게 바꾸거나  
    상품을 장바구니에 유도하기 위해서 리뷰/평점 노출을 더 강화시키는 전략  
2. 다른 요일에 비해 일요일에 조회 수 및 구매 수가 함께 증강하는 경향을 보였기 때문에
    일요일에 프로모션이나 추천상품과 같은 고객들의 눈에 끌리게하는 것들을 더 노출 강화시키는 전략  
3. 카테고리별로 구매 전환 구조가 많이 차이나기 때문에 세그먼트별 맞춤형 전략들을 사용
